# アラート

アラートには以下の二種類の方法が存在する

- クラウドに対して OpenAPI 経由で送信するもの (アラートと呼ばれるもの)
- SendGrid の API 経由で送信するもの (メールアラートと呼ばれるもの)

## アラートの送信間隔

|                            | アラート送信間隔(クラウド)                                           | アラート送信間隔(メール)                                                                                         |
| -------------------------- | -------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------- |
| 8-1 ~ 8-3 センサー監視     | 異常検知から、`ルール` -> `センサー監視条件` の異常継続時間経過毎    | クラウドと同じ                                                                                                   |
| 8-4 通信接続（各センサー） | 異常検知から 5 分毎に 1 回                                           | 異常検知から 5 分後に 1 回、それ以後は異常検知から 24 時間経過毎                                                 |
| 8-5 電池残量               | 異常検知から、`ルール` -> `センサー監視条件` の異常継続時間経過毎    | `マスターデータ` -> `デマンドパルス` に設定された異常継続時間経過後に 1 回、それ以後は異常検知から 24 時間経過毎 |
| 8-6 デマンド監視           | デマンド状況が 警戒、限界、超過、復帰 のどれかに切り替わった時に一回 | クラウドと同じ                                                                                                   |
| 8-7 社屋停電               | 停電発生または UPS 異常発生時(UPS への電源供給が途絶えた) に 1 回    | クラウドと同じ                                                                                                   |
